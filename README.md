# Mini Community

**심심해서 구현해본 커뮤니티 사이트**  

Next.js 16 + React 19를 기반으로 한 커뮤니티 프로젝트로, 게시글 작성, 댓글, 좋아요, 알림 등 기본적인 커뮤니티 기능을 구현했습니다.  
Zustand, Supabase, React Query, TailwindCSS 등을 활용하여 상태 관리, 서버 연동, UI를 구성했습니다.

---

## 🛠️ 기술 스택

- **프레임워크**: Next.js 16, React 19
- **스타일링**: TailwindCSS 4, Framer Motion
- **상태 관리**: Zustand
- **데이터 관리**: React Query
- **백엔드/DB**: Supabase
- **기타**: TypeScript, ESLint, PostCSS

---


## 🚀 주요 기능

- 게시글 작성/수정/삭제
- 댓글 작성/삭제
- 좋아요, 북마크 기능
- 무한 스크롤 리스트
- 알림 기능
- 사용자 인증 및 프로필 관리
